# 🚀 Deviathon 2.0 - Frontend Assignment

A responsive, high-performance landing page for the Deviathon 2.0 hackathon.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom variables, Flexbox, and Grid for responsiveness.
- **Vanilla JavaScript:** DOM manipulation for the countdown and theme toggle.
- **FontAwesome:** Scalable vector icons.

## ✨ Key Features
- **Real-time Countdown:** Dynamic timer using JS `setInterval`.
- **Professional Dark Mode:** Implemented using CSS variables and data attributes.
- **Interactive FAQ:** Custom-built accordion logic.
- **Performance Optimized:** No heavy frameworks, ensuring nearly instant load times.

## 🧠 My Approach
I chose a **Vanilla JS** approach to demonstrate a deep understanding of browser fundamentals and DOM manipulation. By avoiding frameworks for this single-page requirement, I ensured maximum speed and eliminated unnecessary build-step complexity.

## 📱 Responsiveness
The design is fully responsive, using a "Mobile-First" strategy with CSS Grid for the sponsors section and Flexbox for the navigation layout.
